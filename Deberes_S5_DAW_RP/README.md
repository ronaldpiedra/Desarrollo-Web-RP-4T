# Galería Interactiva (Semana 5)

Aplicación web que permite:
- Agregar imágenes mediante una URL.
- Seleccionar una imagen con clic (solo una a la vez).
- Eliminar la imagen seleccionada del DOM.

## Tecnologías
- HTML5
- CSS3 (Grid + Responsive)
- JavaScript (DOM + Events)

## Funcionalidades
- **Agregar:** escribe una URL y presiona el botón o ENTER.
- **Seleccionar:** clic sobre una imagen para resaltarla.
- **Eliminar:** botón “Eliminar Imagen Seleccionada” o tecla Delete/Backspace.
- **Quitar selección:** tecla ESC.
